# Rational Expression Interactive

Vite + React SPA for simplifying and operating on rational expressions with step-by-step UI.

**Live:** [https://content-interactives.github.io/rational_expression/](https://content-interactives.github.io/rational_expression/)

Curriculum alignment: [Standards.md](Standards.md).

## Stack

- React, Vite
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|--------|-------------|
| `npm run dev` | Dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview `dist/` |
| `npm run lint` | ESLint |
| `npm run deploy` | GitHub Pages deploy |

## Configuration

`vite.config.js`: `base: '/rational_expression/'`.

Source: `src/`. Static files: `public/`.
